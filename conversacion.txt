  ### 2. Cumplimiento de Reglas Críticas de Base de Datos

  • Tu Prompt:
  │ "1. La Vista de Totales por Campaña. Lo que NO debes hacer: Traer todas las donaciones a Node.js con un Donation.findAll() y
  usar
  │ un .reduce() o un bucle... Cómo cumplir usando el ORM: Debes crear la vista directamente en tu base de datos (con el comando
  │ CREATE VIEW)...
  │ 2. Transacciones Estables para Transferencias. Lo que NO debes hacer: Hacer un Donation.create() para restar fondos y luego otro
  │ Donation.create() para sumarlos de forma aislada... Cómo cumplir usando el ORM: Debes agrupar ambas operaciones dentro de una
  │ transacción explícita..."

  • Mi Respuesta y Acciones:
  Revisé y verifiqué que el sistema cumpliera al 100% con estas dos restricciones técnicas:
      • Vista SQL ( CampaignTotals ): Inyecté la vista física usando  CREATE VIEW  en el archivo de inicio index.js. Las
      agregaciones de sumas ( SUM ) y conteos ( COUNT ) se delegan por completo al motor SQLite. Se mapeó al modelo pasivo de solo
      lectura CampaignTotals.js.
      • Transacciones ACID: En el controlador de transferencias se implementó una transacción explícita ( sequelize.transaction() ).
      El sistema bloquea los registros dentro de la transacción, lee el balance en tiempo real a través de la vista SQL, valida si
      hay saldo suficiente y, en caso positivo, registra el movimiento. Si algo falla o no hay saldo, ejecuta un  rollback
      inmediato, asegurando que el dinero no se duplique ni desaparezca.

  ### 5. Concepto y Funcionamiento de SQLite

  • Tu Prompt:
  │ "database.sqlite cual es la funcionalidad de este archivo?"

  • Mi Respuesta y Acciones:
  Expliqué detalladamente la naturaleza de SQLite:
      • Es una base de datos embebida en un solo archivo binario en disco.
      • No necesita un servidor corriendo en segundo plano (como MySQL o Postgres).
      • Contiene de manera autónoma toda la estructura DDL (tablas, vistas, claves foráneas) y la información del sistema,
      garantizando portabilidad absoluta (basta con copiar el archivo para trasladar el sistema completo con sus datos).
